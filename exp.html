Graphs and Data Representation Project

Wrote 6 programs to represent undirected and weighted directed graphs and perform operations on said graphs, as well as develop a deeper understanding of data representation by coding 4 programs that perform conversions related to IEEE-754 floating point format and binary represention of numbers. The concepts implemented in the programs include:

  • Undirected Graph Representation
  • Weighted Directed Graph Representation
  • Breadth-first Search
  • Depth-first Search
  • Single-source Shortest path in a Directed Acyclic Graph
  • Dijkstra's Algorithm
  • Conversion to Two's Complement Representation
  • Decimal Fraction to Canonical Binary Fraction
  • Decimal to IEEE-754 FP with Rounding
  • Hexadecimal Bit-pattern in IEEE-FP Format to Decimal Fraction

Cache Simulator

Wrote programs to simulate both a one level and two level computer cache. These programs take in the parameters of the cache including the cache size, the associativity of the cache, the cache replacement policy, and the block size of the cache, as well as a series of memory addresses. The programs would then output the number of memory reads, memory writes, cache hits, and cache misses of each cache.
